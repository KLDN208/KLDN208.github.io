Python3 环境准备

本文针对 macOS 系统

  1. Mac 系统自带 2.* 版本的 python

  2. 安装包管理工具 - Homebrew

  3. 安装 Python3

  4. 编辑器
    PS: 人们可能习惯把 python 和 pycharm 联系起来，但此处我们使用更好的 vscode

    1. 首先，去 vscode 扩展程序 - EXTENSIONS 内安装 Python 扩展程序。
    2. 之前，我们已经安装好 python3 的开发环境。
    3. 最后，我们在 vscode 内创建 .py 结尾的文件，vscode 会自动帮我们切换到 Python 3.9.0 64-bit 环境下开发

  5. 至此，我们的全部前置准备都已结束，你可以开始你的 Python3 之旅了，祝你好运。
